# chuyx_design_pattern
# 设计模式学习
## 设计模式分类：
- 创建型模式
  - 提供了一种在创建对象的同时隐藏创建逻辑的方式，而不是使用 new 运算符直接实例化对象。这使得程序在判断针对某个给定实例需要创建哪些对象时更加灵活。
    - 单例
    - 工厂
    - 抽象工厂
    - 建造者模式
- 结构型模式
  - 关注类和对象的组合。继承的概念被用来组合接口和定义组合对象获得新功能的方式。
    - 适配器
- 行为型模式
  - 关注对象之间的通信。
    - 策略
    - 模板
- J2EE模式
  - 特别关注表示层。这些模式是由 Sun Java Center 鉴定的。
